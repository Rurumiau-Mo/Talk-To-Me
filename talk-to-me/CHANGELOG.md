# Changelog

## 1.0.0 — First Official Release

TalkToMe 1.0.0 is the first stable public release.

### Core features

- Speech bubbles from custom text and RollTables
- Ordered RollTable conversations with up to five NPCs
- Click, token-enter, token-exit, manual, and linked activation
- Switch, door, light, trap, teleport, and reset tiles
- Tile visibility and player-vision restrictions
- Per-tile cooldowns and single-use tiles
- Current-scene tile manager and template-aware tile editor
- Versioned world and tile-data migration
- Macro and public API support
- Optional compatibility with supported Foundry modules
